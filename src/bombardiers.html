Project Gallery / Bombardiers
Fall 2019

As a capstone project for one of my Computer Science courses, I developed a turn-based physics simulation game in Python called Bombardiers.

Based off of an old iOS game, the objective of this turn-based strategy game is to destroy the opponents base by launching missiles across an asteroid field. Whoever destroys the opponent's forcefield first is the winner.



This game has three modes: multiplayer, single player, and a bonus sandbox mode. Each level contains different orientations and sizes of planets, whose gravitational fields affect the trajectory of the projectiles. To allow for a single player mode, I programmed a game AI system to simulate the opponent.

The source code for this project can be found on my Github.
